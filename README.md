# 🩺 Cancer Patient Survival Analysis Dashboard

📊 Interactive Power BI dashboard designed to analyze cancer patient outcomes, survival trends, disease progression, and clinical characteristics through dynamic visualizations and KPI monitoring.
## Key Insights

✔ Survival months decrease significantly in advanced cancer stages

✔ Patients diagnosed at earlier stages show better survival outcomes

✔ MSI status demonstrates noticeable differences in survival patterns

✔ Cancer type distribution highlights the most common diagnosed cancers

✔ Vital status analysis provides a clear overview of patient outcomes
